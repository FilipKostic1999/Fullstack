Fullstack App (Render)

Live demo: https://fullstack-1-0hst.onrender.com/

---

Om projektet
Detta är en enkel fullstack-applikation med frontend, backend och databas som kommunicerar med varandra.  
Frontend (HTML + JavaScript) hämtar data i JSON-format från backend (Express), som i sin tur läser från en SQLite-databas.  
Resultatet visas som en lista med frukter direkt i webbläsaren.

---

Vad är ett CDN?
Ett Content Delivery Network (CDN) är ett globalt nätverk av servrar som levererar statiskt innehåll (t.ex. bilder, script, CSS) från den server som ligger närmast användaren.  
Det gör att webbplatser laddas snabbare, blir mer stabila och klarar högre trafikbelastning.  

Exempel på CDN-tjänster:
Cloudflare, Akamai, Amazon CloudFront.

---

Vad är monitorering?
Monitorering betyder att övervaka en applikations prestanda, tillgänglighet och hälsa i realtid.  
Det används för att upptäcka problem tidigt, analysera belastning och säkerställa att allt fungerar som det ska.  

Exempel på verktyg:
Render Metrics, Datadog, New Relic, Grafana.
